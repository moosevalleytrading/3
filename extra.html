WURFL Capabilities

Understanding device capabilities helps us provide the best experience for our users.

Capability Description Example Use
is_mobile Detects if the device is a mobile device Optimize layouts and features specifically for mobile users
screen_width Physical width of the device's screen in pixels Serve appropriately sized images and adjust layouts
pointing_method Primary input method of the device Adapt interface for touch, mouse, or other input methods
